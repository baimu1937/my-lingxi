# 灵犀工作台 - 功能增强包

## 📦 包含内容

本增强包为灵犀工作台添加了以下核心功能：

1. **全局搜索系统** - Ctrl+F 快速查找所有内容
2. **批量操作管理** - 多选删除、导出等功能
3. **版本控制系统** - 自动保存历史版本，支持回退
4. **增强快捷键** - 15+ 专业级快捷键
5. **拖拽排序** -  intuitive 的列表重排体验

## 🚀 快速开始

### 方法一：直接集成（推荐）

1. 将 `enhancements.js` 复制到 `my-lingxi/` 目录
2. 在 `index.html` 的 `</body>` 标签前添加：

```html
<!-- 增强功能 -->
<script src="enhancements.js"></script>
```

3. 刷新页面即可使用

### 方法二：手动合并

如果你希望完全控制代码，可以将 `enhancements.js` 中的类定义复制到 `index.html` 的 `<script>` 标签内。

## ✨ 功能详解

### 1. 全局搜索 (Ctrl+F)

**触发方式：** 按下 `Ctrl+F` 或 `Cmd+F`

**功能特性：**
- 🔍 实时搜索对话、书稿、提示词、例文
-  高亮显示匹配内容
- 📊 按相关性排序结果
- ⚡ 点击结果直接跳转

**使用场景：**
- 快速找到某次AI对话
- 检索特定书稿片段
- 查找某个提示词的完整内容

### 2. 批量操作

**进入方式：** 
- 长按任意书稿卡片（移动端）
- 右键菜单选择"批量管理"（桌面端，待实现）

**支持操作：**
- ✅ 多选项目
- 🗑️ 批量删除
- 📦 批量导出
- ✕ 清空选择

**退出方式：** 点击工具栏的"取消"按钮

### 3. 版本控制

**自动保存：** 每次保存书稿时自动生成版本快照

**查看版本：**
```javascript
// 在控制台执行
versionControl.showVersionPanel(bookId);
```

**恢复版本：**
- 点击版本列表中的任意版本
- 确认后自动恢复到该内容

**版本限制：** 每本书最多保留50个历史版本

### 4. 增强快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Shift+N` | 新建书稿 | 快速创建新书稿 |
| `Ctrl+Shift+S` | 另存为 | 将草稿保存为新文件 |
| `Ctrl+E` | 切换分屏 | 开启/关闭编辑器分屏 |
| `Ctrl+D` | 复制段落 | 复制光标所在段落 |
| `Ctrl+J` | 合并段落 | 减少多余空行 |
| `Alt+↑` | 上移段落 | 移动当前段落到上一行 |
| `Alt+↓` | 下移段落 | 移动当前段落到下一行 |
| `F2` | 重命名 | 重命名当前打开的书稿 |
| `Ctrl+H` | 历史记录 | 打开历史面板（开发中） |
| `Ctrl+P` | 提示词 | 聚焦到提示词选择框 |
| `Ctrl+Z` | 撤销 | 撤销草稿区的修改 |
| `Ctrl+Y` | 重做 | 重做已撤销的操作 |

**提示：** 所有快捷键均可在 `enhancements.js` 的 `EnhancedShortcuts` 类中自定义

### 5. 拖拽排序

**使用方法：**
1. 鼠标悬停在书稿卡片上
2. 按住左键拖动
3. 移动到目标位置释放

**视觉反馈：**
- 拖动时卡片半透明
- 目标位置显示插入指示线
- 松开后自动保存新顺序

## 🛠️ 高级定制

### 修改搜索范围

编辑 `enhancements.js` 中的 `GlobalSearch.updateIndex()` 方法：

```javascript
updateIndex() {
    // 添加新的索引类型
    this.index.custom = customData.map(...);
}
```

### 调整版本数量

修改 `VersionControl` 构造函数中的参数：

```javascript
constructor() {
    this.maxVersionsPerBook = 100; // 改为100个版本
}
```

### 自定义快捷键

在 `EnhancedShortcuts` 类的 `shortcuts` 对象中添加：

```javascript
this.shortcuts = {
    'Ctrl+K': () => this.customFunction(),
    // ... 其他快捷键
};
```

## 🐛 故障排除

### 问题1：快捷键不生效

**原因：** 与其他应用快捷键冲突  
**解决：** 检查浏览器扩展或其他软件的快捷键设置

### 问题2：搜索无结果

**原因：** 索引未更新  
**解决：** 刷新页面重新建立索引

### 问题3：拖拽无效

**原因：** 浏览器不支持Drag API  
**解决：** 升级到最新版Chrome/Firefox/Edge

### 问题4：版本丢失

**原因：** localStorage被清除  
**解决：** 定期使用"导出备份"功能
